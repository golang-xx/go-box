# go-box
An small bootstrap box for golang HTTP project

## layout
```
main.go            - 工程入口
config.json        - 配置文件
api/               - HTTP API
core/              - 业务逻辑
common/            - 通用全局代码
test/              - 测试目录
web/               - 前端资源

Dockerfile         - Docker 构建文件
control           - 帮助脚本
```
